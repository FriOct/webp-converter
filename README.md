# NEF·JPG → WEBP 변환기

브라우저에서 바로 이미지를 WEBP로 변환하는 정적 웹 페이지입니다. NEF(니콘 RAW)는 파일 안에 내장된 미리보기 JPEG(가장 큰 것)를 찾아 사용하고, JPG·PNG 등은 그대로 불러옵니다. 서버로 파일을 업로드하지 않고 Canvas API를 이용해 브라우저 안에서만 처리합니다.

## 기능

- NEF(니콘 RAW), JPG, PNG 등 다양한 이미지 업로드 (드래그 앤 드롭 또는 클릭)
- 품질 슬라이더로 WebP 압축률 직접 조정 (기본 92)
- 장축(긴 변) 제한으로 리사이즈 (기본 4000px, 0이면 원본 해상도 유지)
- 항목별 다운로드 / 삭제
- 전체 ZIP 일괄 다운로드 (순수 JS로 구현, 무압축 저장 방식)
- 디시인사이드 갤러리 업로드 팁 포함 (849px 강제 축소 회피, 업로드 용량 제한 등)

## 로컬 실행

정적 파일이라 별도 빌드 없이 바로 열립니다.

```bash
open index.html
```

또는 로컬 서버로:

```bash
python3 -m http.server 8000
```
